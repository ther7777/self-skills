# self-skills

个人经验沉淀 / 自用 Codex Skills 分享仓库。

这些 skills 来自我在日常学习、求职准备与远程开发中的可复用流程总结：能减少重复劳动、降低遗漏步骤的概率，并让 Codex 在不同任务间保持一致的工作节奏。

## Included

- `skills/copilot-rule`：对话连续性规则；每个阶段完成后主动给出下一步选择，避免“任务做完就突然结束对话”。
- `skills/tex-resume-validator`：LaTeX 简历编辑的固定闭环；任何 `.tex` 改动后强制执行编译、查错、看 PDF、清理并汇报。
- `skills/resume-optimizer`：算法/AI 工程岗位（校招/实习）简历条目优化 SOP；把原始笔记/逐字稿/项目描述提炼成可量化、可面试追问的 bullet。
- `skills/ssh-remote-access`：在修改远程代码或跑实验前，建立并验证到 Linux 服务器的 SSH 访问流程。

## Notes

- 这些是“自用流程沉淀”，不保证适用于所有团队/项目；欢迎按需裁剪。
- 仓库结构遵循 `skills/<skill-name>/SKILL.md`。
